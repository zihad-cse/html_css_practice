Usually when a browser window is shrunken, the content within the flexbox parent would shrink as much as needed.
But as seen above, the content inside the parent flexbox wraps around and shifts onto different lines when the parent div gets smaller, that is because of the [flex-wrap: wrap;] property that was applied to the parent flexbox.
