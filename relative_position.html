Relative Position

The element is positioned according to the normal flow of the document, but can be offset relative to itself based on the values of top, right, bottom and left values. The offset does not affect anything else in the page.


The red box has the Relative position, But it was offset using the top, right, bottom and left values. The box didn't affect the sample text that was written on top.
