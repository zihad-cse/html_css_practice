Aligning content within a flexbox can be simplified further. We can imagine there are two axes. The "Main Axis" and the "Cross Axis".
The Main Axis goes through the middle of the containing div horizontally. Whereas the cross axis goes through the middle vertically.
In CSS. The [align-items] attribute is the cross axis. So if content needs to be aligned vertically, that attribute is used. And [justify-content] Is the main axis, so if content needs to be aligned horizontally that attribute is used.
Above, an example is shown where the children boxes are aligned vertically to start from the top using the [align-items: flex-start] attribute in the container div. and it is aligned horizontally to have space around the children div with the Main Axis attribute [justify-content: space-around].
