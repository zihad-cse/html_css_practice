Just as the flexbox properties of the containing flexbox can be manipulated. The children flexboxes of the parent flexbox can be manipulated as well using flex attributes.
Above, are a few of the attributes. The [flex-shrink: 0;] makes it so that the flexbox does not shrink below it's original size when the viewport is shrunken.
[flex-grow: 1;] makes the flexbox grow to fill up any remaining space there is in the parent box.
and [flex-grow: 2;] makes sure that the box grows at least twice as much or more than the flexbox with the value set as 1.
To make it grow exactly twice as much as the box with [flex-grow: 1;] we need to add the [flex-basis: 0;] property. What that'll do is that will make the box imagine that it's starting off from 0 width and then makes it exactly double. The example for it is given below.

The alignment of the items can be overridden and can be changed individually using the [align-self] property. Whichever child flexbox has this property will override the parent's alignment rules and align itself based on the stated rules in self-align.
In the example above, "Child 2" has the property of [self-align: center;] which made it ignore the [align-items: stretch;] property set in the parent box and centered itself.
