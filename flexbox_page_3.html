In the parent container shown above. The children boxes have been spaced in a way where the spare space around them have been distributed between them instead. this was achieved using the [justify-content: space-between;] attribute in css.
similarly, spare space can be distributed evenly between them using [justify-content: space-even;] or space can be distributed in a manner where the space is also distributed around the children as well, using the [justify-content: space-around;] attribute for the container div.
